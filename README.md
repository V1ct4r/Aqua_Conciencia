# AquaConciencia
Plataforma web educativa sobre la escasez hídrica que esta aconteciendo en el archipielago de Chiloé. 
El principal objetivo de esta plataforma es generar conciencia ambiental, principalmente a los estudiantes de 5 a 8 básico, ya que la web esta desarrollada para ser sumamente interactiva y didactica, viendose esto reflejado en el mapa, quiz, sistema de progresion al completar quiz, entre otras.

## Tecnologías usadas
-Python
-Django
-HTML
-CSS
-JavaScript

### Avances hasta el momento
-Registro de usuarios
-Creacion de cuenta
-Restableción y recuperación de contraseñas
-Visualización de la pagina principal (mapa, noticias, quiz, ayuda, entre otras)
-Redirección al seleccionar botones (noticias, curiosidades, quiz, educacion ambiental, logo del perfil y logo de la pagina principal)

#### Como ejecutarlo
1. Clonar el repositorio: git clone https://github.com/V1ct4r/Aqua_Conciencia.git
2. Activar el entorno virtual (ven): .\venv\Scripts\Activate.ps1
3. Instalar dependencias: pip install django
4. Correr el servidor: python manage.py runserver
5. (opcinal) Abrir la url en el navegador de preferencia: http://127.0.0.1:8000/
